Theming

Set data-theme="minimal" on <html>. Theme IDs: minimal, brutalist, glass, neon, corporate, editorial, playful, luxury, retro, organic.

Edit src/themes/<id>/tokens.css. Add themes using src/docs/templates/new-theme.tokens.css.

Light and dark: each theme has [data-theme="id"] and [data-theme="id"].dark blocks. Color mode via next-themes.

Deployment

Deploy to Vercel, Netlify, or npm run build && npm run start on Node 20+. Set NEXT_PUBLIC_SITE_URL. Use production URL as CodeCanyon live preview.
