## FAQ entries (CodeCanyon FAQ tab)

### Are these themes just color swaps?

**No.** Each of the 10 themes defines fonts (`--font-heading-family`), radius (`--radius`), shadows, borders, and theme-specific signature styles — not only OKLCH color tokens. Open `/capture/theme-identity-grid` on the live demo to see the **same dashboard** in all 10 identities. Brutalist uses 0px radius and hard offset shadows; Playful uses large pill radii; Editorial and Luxury use serif display type.

### What license do I need?

- **Regular** — one end product, end users not charged (client work, free products).
- **Extended** — one end product where end users pay (subscription SaaS, paid app).

You may not redistribute the source as a competing UI kit. See `LICENSE-NOTES.md`.

### How do I get support?

Post in **item comments** with Node version, package manager, and reproduction steps. For documentation, see `docs-buyer/Documentation.md` (in the ZIP) or `/docs` on the live demo.

### Do I get updates?

Yes — **lifetime updates** through your CodeCanyon downloads page when new versions are published.

### What stack is required?

Node.js **20+**, **Next.js 16**, **React 19**, **Tailwind CSS v4**, **TypeScript**. The demo runs without external APIs or database setup.

### Where is the live preview?

Use the URL in the item description. Recommended paths:

- `/` — marketing homepage with auto-cycling hero
- `/capture/theme-identity-grid` — identity comparison grid
- `/preview` — full demo hub (9 pages, 10 themes)

### Can I use this for client projects?

Yes, with the appropriate license (Regular for free client deliverables, Extended if the client product charges end users).
